# dhanudeep

This is the backend REST API for the gitlite cli, analytics and insights tool for glc telemetry data.

## Installation

### first clone the repository

```bash
git clone https://github.com/NikhilKatkuri/glc_docs.git
cd glc_docs
```

### run cmds

```bash
git checkout -b backend
git status
```

output:

```bash
On branch backend
nothing to commit, working tree clean
```

### if you want to push to this branch

```bash
git add .
git commit -m "your commit message"
git push origin backend
```
